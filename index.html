About ClassCord

Classcord is a bot developed with the purpose of facilitating the use of discord in an educational environment

The bot is designed to be as simple as possible for both teachers and students

Features

Great level of simplification to facilitate the use of the bot and the server for people who do not understand much about discord

Verification system to easily assign each user their role on the server

Reminder system so that students do not forget any task

Event system so that teachers can schedule exams, assignments, readings, etc

Study time tracking system

Pomodoro system to improve productivity

Question system that students can use to ask their questions to teachers

Commands to provide the best discord calling experience for online classes

Commands

  • Command
    Function
  • .changeprefix {prefix} | only for the owner
    Change the prefix of the server (If this command doesn't work for you, add classcord to your server again)
  • .addteacher @member | only for teacher or admin
    Get a teacher role for the member
  • .removeteacher @member | only for teacher or admin
    Remove the teacher rol for the member
  • .ask {question}
    Use this command to ask your teacher your question
  • .remind {time to reminder} {content}
    Use this command to create reminders. For example: .remind 12m example or .remind 12h example or .remind 12d example
  • .checkreminds
    Use this command to check the reminds (if there are no scheduled reminds the command will not return anything)
  • .create {event date} {title} | only for teachers
    Use this command to create events like exams. For example: .create 12m exam or .remind 12h exam or .remind 12d exam
  • .checkevents
    Use this command to check the events if there are no scheduled events the command will not return anything)
  • .deleteevent | only for teachers
    Use this command to delete events
  • .pomodoro
    Use this command to use the pomodoro system to improve productivity
  • .start {subject} | the subject is optional
    Use this command to start the study timer
  • .stop
    Use this command to start the study timer and get your study time
  • .mute @member | only for teachers
    Use this command to mute a specific student
  • .unmute @member | only for teachers
    Use this command to unmute a specific student
  • .muteall | only for teachers
    Use this command to mute all students
  • .unmuteall | only for teachers
    Use this command to unmute all students
